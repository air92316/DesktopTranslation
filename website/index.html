功能特色

⌨

雙擊 Ctrl+C

不打斷工作流程，選取文字後連按兩次 Ctrl+C 即可翻譯。第一次正常複製，第二次觸發翻譯。

🌐

多引擎支援

預設使用免費的 Google Translate，進階可切換 Claude 或 OpenAI，支援 13 種語言。

🎨

日式極簡設計

深淺色主題自動切換、精緻動畫效果、4K 螢幕完整支援，視覺清爽不干擾。

🔒

隱私安全

API Key 使用 Windows DPAPI 加密、所有翻譯本地處理、不上傳個人資料。
