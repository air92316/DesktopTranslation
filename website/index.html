Windows SmartScreen 提示？

首次執行時，Windows Defender SmartScreen 可能會顯示「已防止某個無法辨識的應用程式啟動」。 這是因為程式尚未取得數位簽章，並非惡意軟體。

解決方式：點擊「更多資訊」→「仍要執行」即可正常啟動。

功能特色

⌨

雙擊 Ctrl+C

不打斷工作流程，選取文字後連按兩次 Ctrl+C 即可翻譯。第一次正常複製，第二次觸發翻譯。

🌐

多引擎支援

預設使用免費的 Google Translate，進階可切換 Claude 或 OpenAI，支援 13 種語言。

🎨

日式極簡設計

深淺色主題自動切換、精緻動畫效果、4K 螢幕完整支援，視覺清爽不干擾。

🔒

隱私安全

API Key 使用 Windows DPAPI 加密、所有翻譯本地處理、不上傳個人資料。

🔄

自動更新

應用程式內建自動更新，偵測新版本後一鍵下載安裝。下載驗證確保安全，全程中文介面。

使用教學

安裝完成後，按照以下步驟開始使用

1

找到系統匣圖示

安裝完成後，程式會在右下角系統匣（工具列）顯示一個藍色「T」圖示。如果看不到，請點擊工具列的 ^ 展開箭頭。

T ← 找到這個圖示
2

開始翻譯

在任何地方選取文字，然後快速按兩次 Ctrl+C（間隔 0.4 秒內），翻譯視窗會自動彈出並顯示翻譯結果。

Ctrl + C → Ctrl + C = 翻譯！
3

右鍵選單

右鍵點擊系統匣圖示，會出現快捷選單：

顯示主視窗
翻譯引擎 ▸
☑ 開機自啟動
⚙ 設定
🔄 檢查更新
關於
結束
4

設定詳細說明

點擊右鍵選單的「設定」開啟設定視窗，以下是各項設定說明：

翻譯引擎
Google（預設）— 免費、無需設定，適合日常翻譯
LLM — 使用 AI 模型翻譯，品質更高，需要 API Key
LLM Provider
選擇 AI 翻譯提供者：
Claude（Anthropic）— 前往 console.anthropic.com 註冊並取得 API Key
OpenAI — 前往 platform.openai.com 註冊並取得 API Key
API Key
輸入你的 LLM API Key。此金鑰會以 DPAPI 加密儲存在本機，不會上傳到任何伺服器。
僅使用 Google 翻譯時無需填寫此欄位。
開機自啟動
開啟後，Windows 啟動時會自動在系統匣載入 DesktopTranslation，隨時可用雙擊 Ctrl+C 翻譯。
雙擊間隔
兩次 Ctrl+C 的最大間隔時間（毫秒）。預設 400ms。
如果經常誤觸，可以調低到 300ms；如果反應不及，可以調高到 500ms。
TTS 語速
文字轉語音的朗讀速度。1.0 為正常速度，0.5 為慢速，2.0 為快速。
點擊翻譯視窗中的 🔊 按鈕可朗讀原文或翻譯結果。
主題
跟隨系統（預設）— 自動配合 Windows 深色/淺色模式
淺色 — 永遠使用淺色主題
深色 — 永遠使用深色主題
5

翻譯視窗操作

⇄ 互換
點擊語言列中間的互換按鈕，將翻譯結果移到左邊並反向翻譯
📌 置頂
點擊標題列的圖釘按鈕，讓翻譯視窗永遠在最上層
語言下拉選單
自動偵測不準確時，可以手動選擇來源語言和目標語言（支援 14 種語言）
📋 複製
一鍵複製翻譯結果到剪貼簿
🔊 朗讀
左右兩邊各有朗讀按鈕，使用系統 TTS 語音朗讀文字
歷史紀錄
點擊底部的「翻譯歷史」展開最近 50 筆翻譯紀錄
